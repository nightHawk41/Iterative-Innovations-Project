## Creating a Branch Steps:
1. Navigate to the GitHub repository for the project.
2. Click on the "Branch: main" dropdown menu.
3. Type the name of your new branch (e.g., "feature/new-feature") in the search box.
4. Click on "Create branch: feature/new-feature" to create the new branch.
5. You can now switch to your new branch and start working on your assigned tasks.

## Pull Request Steps:
1. After completing your work on the new branch, navigate to the GitHub repository.
2. Click on the "Pull requests" tab at the top of the page.
3. Click on the "New pull request" button.
4. Select the base branch (e.g., "main") and the compare branch (your new branch).
5. Review the changes and click on "Create pull request."
6. Add a descriptive title and comment for the pull request, then click "Create pull request" to submit it for review.
